12年9月份帮朋友做的一个仿google+的分享新鲜事的web应用
今天用了点时间，抽出一部分共享

### 　
前端：html5+css3+js
后端：php+mysql
兼容：Google Chrome、Mozilla Firefox、ie7+浏览器

### 功能有：
可编辑div
上传图片、摄像头拍照及人性化裁剪和缩略
添加视频、抓取优酷、酷6视频及截取封面图
上传附件
抓取网页快照
双击输入框获取cookie中的内容草稿
输入框自动过滤html
输入框自适应内容高度
锁定评论锁定消息

### 使用
1、将doc下的fenxiang.sql导入到数据库
2、修改config.inc.php中的数据库配置信息
3、uploads权限需要可写
4、测试
5、详细使用说明参考doc目录下：operation-manual.doc
